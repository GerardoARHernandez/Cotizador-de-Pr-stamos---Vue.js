# Cotizador de Préstamos - Vue.js

Este es un proyecto de **Cotizador de Préstamos** desarrollado con **Vue.js**. La aplicación permite a los usuarios calcular el monto de un préstamo en función del plazo y la cantidad seleccionada.

## Características

- Selección de cantidad y plazo del préstamo
- Cálculo dinámico de pagos
- Interfaz simple y fácil de usar
- Implementación de componentes reutilizables en Vue.js
- Estilos con TailwindCSS

## Tecnologías Utilizadas

- **Vue.js**: Biblioteca principal para construir la interfaz de usuario.
- **TailwindCSS**: Framework para la creación de estilos de manera rápida y eficiente.
- **JavaScript (ES6+)**: Para manejar la lógica del cotizador.

## Uso

    Ingresa la cantidad que deseas solicitar.
    Elige el plazo en meses para pagar el préstamo.
    Obtén el cálculo instantáneo de la cantidad total a pagar, incluyendo los intereses.
